# Calculator By Sneha

A beginner-friendly calculator project developed using Python.  
This calculator can perform both basic and scientific calculations.

## Features

### Basic Calculator
- Addition
- Subtraction
- Multiplication
- Division

### Scientific Calculator
- Sin
- Cos
- Tan
- Logarithm
- Square Root
- Power
- Pi Value

## Technologies Used
- Python
- Math Module

## How to Run

1. Open the project folder
2. Run the Python file

```bash
python calculator.py
